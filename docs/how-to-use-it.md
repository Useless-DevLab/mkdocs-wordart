# How to use it

To use this beautiful plugin, you just have to put this MarkDown inside your docs :

```
 ```wordart 
 {
     "style": "rainbow",
     "text": "rainbow",
     "size": 70
 }
 ```
```

`style` : the style you want for your WordArt.

All avaible styles : 

- rainbow
- green-marble
- horizon
- blues
- sunset
- superhero
- graydient
- outline
- italic-outline
- slate
- radial
- paper-bag
- chrome
- up
- purple
- marble-slab
- mauve
- tilt
- yellow-dash
- gray-block
- aqua
- red-blue
- basic-stack
- brown-stack
- texture-stack
- green-stack
- stack-3d

`text` : The text you want to display

`size` : Size of the Word, default is 50.
